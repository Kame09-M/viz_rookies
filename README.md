# Viz Rookies 2025 - データ分析プロジェクト

**テーマ：「近いから、離れる。〜福島県における若年女性の転出構造と定住促進策〜」**

チーム：まさひこ・さきな

---

## プロジェクト概要

福島県における若年層（特に女性15〜34歳）の転出超過問題を多角的なデータで分析し、
「就職先として福島を選べる仕組み」を提案するプロジェクトです。

### 加工済みデータ（Tableau用）

#### `idou_tableau.csv`
- **目的：** `a00301s.xlsx` `a00302s.xlsx` `a00303s.xlsx` の3ファイルをTableauで使いやすい縦持ち形式に変換
- **加工内容：**
  - 横持ち（年齢が列）→ 縦持ち（年齢が行）に変換（アンピボット）
  - 3ファイルを1つに統合
  - 国籍コード60000（移動者全体）のデータのみ残す
- **列：** 種別、都道府県、年次、性別、年齢階級、人数

#### `idou_tableau_v2.csv`
- **目的：** `idou_tableau.csv` の重複データを修正したバージョン
- **加工内容：**
  - 国籍コード60000（移動者）のみに絞り、日本人・外国人の重複行を除去
  - これにより各都道府県のデータが正確に1行になった
- **列：** 種別、都道府県、年次、性別、年齢階級、人数

#### `tenshutsu_all.csv`
- **目的：** 都道府県別・性別・年齢階級別の転出超過率を計算するための汎用データ
- **加工内容：**
  - `idou_tableau_v2.csv` の転入超過数データを使用
  - `b02_03.xlsx` の2020年国勢調査人口データと都道府県をキーに結合
  - 転出超過率（%）= 転出超過数 ÷ 人口 × 100 を計算
  - 47都道府県のみ抽出（広域圏・市区町村を除外）
  - 転出超過の場合はマイナス表記で統一
- **列：** 都道府県、性別、年齢階級、転出超過数、人口、転出超過率(%)
- **用途：**
  - 女性20〜24歳の全国比較（ワースト11位）
  - 女性15〜34歳の全国比較（ワースト9位）
  - 男女比較グラフ

#### `tenshutsu_rate.csv`
- **目的：** 女性20〜24歳に絞った転出超過率の全国比較用データ
- **加工内容：** `tenshutsu_all.csv` から女性・20〜24歳のみ抽出
- **列：** 都道府県、転出超過数、女性20〜24歳人口、転出超過率(%)

#### `fukushima_destination.csv`
- **目的：** 「福島の若者がどこへ転出しているか」を可視化するためのデータ
- **加工内容：**
  - `b00802s.xlsx` から転出元＝福島県のデータのみ抽出
  - 47都道府県のみ（広域圏・市区町村を除外）
  - 国籍コード60000（移動者）のみ
  - 年齢は15〜34歳（15〜19歳・20〜24歳・25〜29歳・30〜34歳）
  - 年齢表記を他CSVと統一（例：`15-19歳` → `15歳～19歳`）
- **列：** 転出先都道府県、性別、年齢階級、転出者数
- **主な発見：** 福島女性15〜34歳の転出先1位は東京都（2,045人）、2位は宮城県（1,249人）

#### `anketo.csv`
- **目的：** 首都圏在住福島県出身若年層のアンケート結果をTableauで可視化するためのデータ
- **出典：** 福島県「首都圏在住福島県出身若年層アンケート調査結果概要」（令和6年実施、n=512）
- **加工内容：** PDFのアンケート結果を手動でCSV化
- **列：** カテゴリ、質問、選択肢、割合(%)
- **カテゴリ：**
  - `転出理由`：県内進学しなかった理由
  - `就職`：県内就職しなかった理由
  - `愛着`：福島県への愛着度
  - `Uターン`：Uターンの可能性
  - `リモート`：Uターン条件・移住定住促進

## 分析の主な発見

```
1. 福島県の女性15〜34歳の転出超過数は全国ワースト4位（-3,415人）
2. 転出は20〜24歳に集中（進学・就職の2段階で流出）
3. 転出先の1位は東京都、2位は宮城県（近距離の大都市へ）
4. 愛着はある（70.5%）でも戻れない（51.8%がUターン予定なし）
5. 転出理由は「給料が低い」「希望職種がない」が上位
```
